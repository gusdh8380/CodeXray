# Intent (MVP)

## Inputs
- 로컬 디렉토리 경로
- Git 레포지토리 URL
- 압축 파일

## Must-have features
1. **구조 분석** — 인벤토리 / 의존성 그래프 / 호출 관계 / 진입점 자동 식별
2. **정량 평가** — 복잡도·중복도·결합도/응집도·문서화·테스트·보안 → A~F 등급
3. **정성 평가 (AI)** — 점수 + 근거 코드 인용 + 개선 제안
   (가독성·설계·유지보수성·위험)
4. **핫스팟 매트릭스** — 사용/변경 빈도 × 품질 점수 2축, 우선 개선 영역 자동 추출
5. **종합 리포트** — 1페이지: 등급, 강점/약점 Top 3, 핫스팟 리스트, 근거 라인 포함 권고
6. **인터랙티브 대시보드** — 그래프 노드 클릭 / 함수·클래스 검색 / 파일별 평가+요약+제안

## Non-goals (이번 버전에서 안 할 것)
- 자동 코드 수정·리팩터링 PR 생성 → **분석/평가만**
- 실시간 모니터링·CI 통합 → **단발성 스캔만**
- 다인 협업·코멘트·이슈 트래커 → **개인 도구**
- SaaS 호스팅 → **로컬 실행 우선**

## Success criteria
- 코드베이스 입력 후 **5분 내** 구조·평가 동시 출력
- 리포트만 보고 "다음에 무엇을 할지" 결정 가능
- Python · Java · JavaScript · C# 동일 인터페이스
- 동일 코드 반복 평가 시 결과 편차가 일정 범위 이내

## Deliverables
- 1페이지 종합 리포트 (Markdown / PDF)
- 인터랙티브 웹 대시보드
- 평가 결과 JSON (재사용·비교)
